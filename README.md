# 四神将棋（fourguardians）

青龍・白虎・朱雀・玄武の **4 陣営**で戦う、オリジナルの 4 人対戦将棋をブラウザで遊べるようにするプロジェクト。
駒の基本の動きは将棋と同じだが、十字型の盤を 4 人で囲んで戦う。

> **考案者のオリジナルゲーム**です。ルールは作者の頭の中にあるため、確定済みの仕様は [`HANDOFF.md`](./HANDOFF.md) に集約しています。新しく開発を引き継ぐときは **まず `HANDOFF.md` を読んでください**。

## 技術スタック

- React 18 + TypeScript + Vite 6

## 動かし方

```bash
npm install
npm run dev      # http://localhost:5173/ で開く
npm run build    # 型チェック + 本番ビルド
```

Node.js 18+ 推奨（開発時は Node 25 / npm 11 で確認）。

## 今できること（2026-06-08 時点）

- 11×11 十字型の盤と 4 陣営の初期配置を表示
- **視点切り替え**：自分の陣を常に手前（下）に表示できる
- 駒をクリック→合法手をハイライト→移動（将棋と同じ動き、各陣の向きに回転）
- 駒を取ると**持ち駒**になり、駒台に表示／持ち駒を**打てる**
- **成り**（中央 3×3 ＋ 敵陣で成れる。任意／強制を判定、成り選択モーダルあり）
- 手番は**時計回り**（手前→左→上→右）に進行
